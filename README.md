# 🧮 GUI Calculator

A full-featured calculator app built with Python and Tkinter, simulating the interface and behavior of a physical calculator.

## 🚀 Features
- Basic Arithmetic: `+`, `-`, `×`, `÷`
- Advanced: `√`, `^`, `%`, `π`
- Bracket support
- Full calculation history
- Buttons for: `AC`, `C`, `Cut`
- Responsive UI with modern design
